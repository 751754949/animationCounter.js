How to use it

1. Installation & Use

Download zip and add animationCounter.js after jQuery, before your closing body tag

<script src="animationCounter.js" type="text/javascript">

or install with bower

bower install animationCounter.js

Then, in your JS file, call the animationCounter() function.

$('yourdiv').animationCounter();

or Call the animationCounter() function with the options

$('yourdiv').animationCounter({
start: 0,
end: null,
step: 1,
delay: 1000,
txt: ' '
});

2. Options

start
The value where the counter started
integer - default: 0

end
The value where the counter stopped
integer - default: null

step
The interval between the numbers
integer - default: 1

delay
The intervals (in milliseconds) on how often to execute the code default
integer - default: 1000

txt
The text displayed after your counter
string - default: ''
